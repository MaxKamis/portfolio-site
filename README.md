## Setup

To run this project locally, follow these steps:

1. Clone the repository:
    ```bash
    git clone https://github.com/maxkamis/portfolio-site.git
    ```
2. Navigate to the project directory:
    ```bash
    cd portfolio-site
    ```
3. Commend "npm run dev" to view the site.
